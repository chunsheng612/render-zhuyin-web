# render-zhuyin-web

Codex Skill for rendering Mandarin Zhuyin / Bopomofo in web pages using the 中文直式注音 display logic.

## Install

Clone this repository into your Codex skills folder:

```bash
cd ~/.codex/skills
git clone https://github.com/chunsheng612/render-zhuyin-web.git
```
